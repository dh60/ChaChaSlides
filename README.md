# MetalSlide
MetalSlide is a minimalist slideshow app for MacOS that uses Metal for high quality viewing of images.

## Features
- Displays photos from a folder of your choice in shuffled order.
- Full Metal 4 pipeline for native rendering without any OS scaling.
- MetalFX upscaling, Lanczos-3 downscaling.
- Can delete photos from within the slideshow.

## Usage
- Start the app.
- Select your folder.
- Right Arrow / Spacebar = Next
- Left Arrow = Previous
- Delete = Delete Photo (to Trash)
- Esc = Exit
